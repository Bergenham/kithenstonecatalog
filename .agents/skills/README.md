# Локальные skills для проекта

Сюда скопированы релевантные skill-пакеты из `D:\Documents\SKILLS\loc`, которые полезны для поддержки этого Django-сайта.

## Подключенные skills

- `django-web-framework` - работа с Django ORM, views, URL и admin.
- `project-architecture-planning` - планирование архитектуры, требований и задач.
- `architecture-decision-records-adr-5` - оформление ADR для важных технических решений.
- `python-code-review-linting-1` - ревью Python-кода, linting, security-антипаттерны.
- `seo-expert` - on-page и technical SEO для страниц сайта.
- `technical-seo-auditor-1783336230014` - аудит robots.txt, sitemap.xml, Core Web Vitals, индексируемости.
- `site-architecture-assistant` - структура сайта, навигация, URL-паттерны и внутренняя перелинковка.

## Не скопированы

Эти skills есть в исходной папке, но не относятся к текущему стеку проекта:

- `advanced-celery-orchestration-1` - Celery в проекте не используется.
- `python-clean-architecture` - skill ориентирован на FastAPI-архитектуру.
- `python-fastapi-mastery` - FastAPI в проекте не используется.
- `sqlalchemy-orm-patterns` - проект использует Django ORM, не SQLAlchemy.
- `telegram-bot-development-1` - Telegram-бота в проекте нет.
